## 1. Background

In our design, the OS includes Docker, kubernetes, and you can get the packages by yourself.
Note that you should first disable selinux and firewalld of your OS.

## 2. All-in-one

you can install all softwares on CentOS/RHEL 7.x using the following script

```
curl https://github.com/kubesys/kubeos/releases/download/1.1/centos7.sh | sh
```
